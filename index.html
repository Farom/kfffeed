kfffeed -  KDE Fritz Fon Feeder


KDE tool to ex- and import Kontacts to a FRITZ!Box Fon WLAN 7270 for usage with a Dect phone.

• What it is

At the Moment you can export Contacts from kaddressbook to a FritzBox.
It will generate an xml file, you have to upload into the FritzBox.

• What it should be

Next step in development is to merge intellegent with the existing entries of the phonebook. After that it becomes a gui.
For all other things  —   look on the website of GitHub
